Chapter 4
Miscellaneous messages

The following messages are supported by all the objects. They are detailled in specialized sections.

miscMsgs
  • eval: evaluates a list of messages in the context of the receiver object. See section 17.2 p.177 for more details.
  • watch: used to manage the object interaction with various events. See section 16 p.159 for more details.
  • push, pop: saves and restores the object interaction state. See section 16.6 p.173 for more details.
  • map: used to describe the relations between graphic and time spaces. See section 12 p.98 for more details.
